# Right Hemisphere — Creative (Task-Directed)

> Synthesis, holistic review, challenge. Fires when the thalamus activates it.

## Purpose

Task-directed creative processing. Operates within the Integration Cycle — divergent scoping (Phase 1) and adversarial review (Phase 3). Not for idle-state association; that's the DMN.

## When It Fires

- Integration Cycle Phase 1: scope the problem space before left hemisphere executes
- Integration Cycle Phase 3: challenge left hemisphere's output — what's missing, what doesn't fit, what would a contrarian say?
- Cross-domain synthesis when corpus callosum requests it
