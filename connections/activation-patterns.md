# Activation Patterns — Frequent Neural Pathways

> Common circuits that activate together. Documented for optimization.

## High-Frequency Pathways

```
Pattern A: Standard Processing (40% of sessions)
  thalamus/inbox.md → cortex/prefrontal/executive.md → cortex/left/ + cortex/right/ → hippocampus/context-state.md

Pattern B: Crisis Mode (10% of sessions)
  thalamus/inbox.md → amygdala/urgent.md → cortex/prefrontal/executive.md → cortex/left/ → hippocampus/gotchas.md

Pattern C: Creative Exploration (25% of sessions)
  cortex/prefrontal/executive.md → cortex/right/ → connections/corpus-callosum.md → cortex/left/ → hippocampus/context-state.md

Pattern D: Deep Analysis (20% of sessions)
  cortex/prefrontal/executive.md → cortex/left/ → connections/corpus-callosum.md → cortex/right/ → hippocampus/gotchas.md

Pattern E: Reflection (5% of sessions)
  hippocampus/context-state.md → hippocampus/gotchas.md → cortex/prefrontal/executive.md → automation/mirror-neurons.md
```

## Anti-Patterns (Avoid)

- cortex/left/ → cortex/left/ (loop without creative check)
- amygdala/ → cortex/right/ (urgent issues need action, not exploration)
- Direct: any → falsificator.md (cleanup needs executive awareness)
